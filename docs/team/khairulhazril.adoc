== Overview

AddressBook - Level 4 is a desktop address book application used for teaching Software Engineering principles. The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and has about 10 kLoC.

== Summary of contributions

* *Major enhancement 1*: *Sort* - Sorts the tasks in the task manager.
** What it does: It allows the user to view the tasks in a more systematic order by name, module, date or priority.
** Justification: This feature improves the product significantly because it provides the user with the ability to organise the task list based on the attribute they choose. This allows the user to plan their schedule based on the due date, priority or module.
